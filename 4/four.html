Ordered List of HTML Tags

  1. <html> - Defines the root of an HTML document.
  2. <head> - Contains meta-information about the HTML document.
  3. <title> - Sets the title of the HTML document displayed in the browser's title bar.
  4. <meta> - Provides metadata about the HTML document, such as character encoding and viewport settings.
  5. <link> - Links external resources like stylesheets.
  6. <style> - Defines embedded CSS styles within the HTML document.
  7. <script> - Embeds or links to JavaScript code.
  8. <body> - Contains the visible content of the web page.
  9. <h1>, <h2>, ..., <h6> - Defines headings of different levels, with <h1> being the highest.
  10. <p> - Represents a paragraph of text.
  11. <a> - Creates hyperlinks to other web pages or resources.
  12. <img> - Embeds images in the web page.
  13. <ul> - Defines an unordered (bulleted) list.
  14. <ol> - Defines an ordered (numbered) list.
  15. <li> - Represents a list item within <ul> or <ol>.
  16. <table> - Defines a table.
  17. <tr> - Represents a table row.
  18. <td> - Defines a table cell.
  19. <form> - Creates an HTML form for user input.
  20. <input> - Represents an input field in a form.
  21. <button> - Creates a clickable button.
